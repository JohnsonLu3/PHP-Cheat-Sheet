Constants and Concatenation


<?php
   define('PI', 3.1415926)

   echo "Pi is " . PI;
?>
define(NAME, VALUE)

To make a constant you use the deine function. Concatenation is preformed with the '.' like shown above.

Casting


<?php
    echo "120 / 8 = " . (integer)(5/2);
?>
(TYPE)(VALUE)

Casting is php is a lot like Python casting. Data type wrapped around () and followed by the value you want to cast also wrapped.
