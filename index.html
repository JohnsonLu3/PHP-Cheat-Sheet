Casting


<?php
    echo "120 / 8 = " . (integer)(5/2);
?>
(TYPE)(VALUE)

Casting is php is a lot like Python casting. Data type wrapped around () and followed by the value you want to cast also wrapped.

Arrays


<?php
    $numbers = array(1,2,3,4,5,6,7);

    $numbers[7] = 8;

    for($i = 0; $i < sizeof($numbers); $i++){
        echo $numbers[i];
    }

    foreach($numbers as $num){
        echo $num;
    }

    sort($numbers)      // sort arrays in ascending order
    rsort($numbers)     // sort arrays in descending order
    asort($numbers)     // sort associative arrays in ascending order, according to the value
    ksort($numbers)     // sort associative arrays in ascending order, according to the key
    arsort($numbers)    // sort associative arrays in descending order, according to the value
    krsort($numbers)    // sort associative arrays in descending order, according to the key

?>
array(value1, value2, ...)

To make an array you must call the array function. These arrays are not type safe.

Maps


<?php
    $customer = array('1289571' => 'John', '1869372' => 'Kim', '1635527' => "David");

    foreach($customer as $key => $value){
        echo 'ID: ' . $key . ' | ' . $value;
    }
?>
array(key1 => value1, key2 => value2, ...)

Creating a map is just an array with key, value pairs. The key, values are seperated by the '=>' operator.

Php with SQL

Connecting


<?php
    $servername = "localhost";
    $username = "username";
    $password = "password";

    // Create connection
    $conn = new mysqli($servername, $username, $password);

    // Check connection
    if ($conn->connect_error) {
        die("Connection failed: " . $conn->connect_error);
    } 
    echo "Connected successfully";

    // Close the connection
    $conn->close();
?>

To establish a connection you'll need the database's name, username, and password. Then create the a connection object using that info. You can check if the connection was successful by checking connect_error found in the connection object.

Submitting Queries


<?php
    $sql = "SELECT id, firstname, lastname FROM customers";
    $result = $conn->query($sql);

    if ($result->num_rows > 0) {
        // output data of each row
        while($row = $result->fetch_assoc()) {
            echo "id: " . $row["id"]. " - Name: " . $row["firstname"]. " " . $row["lastname"]. "
";
        }
    } else {
        echo "0 results";
    }
?>

First define your query statment and the call query on your connection object. This will return a set of results.
